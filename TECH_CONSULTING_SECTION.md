## 🎨 المكونات الرئيسية

### 1️⃣ **خلفية متحركة (Animated Background)**
- خلفية داكنة (#0a0a1e)
- دوائر متوهجة متحركة (برتقالي وبنفسجي)
- تأثير Float animation يدوم 20 ثانية
- Gradient overlay شفاف

### 2️⃣ **Header Section**
- **Badge** متوهج مع أيقونة lightbulb
- **عنوان** كبير مع كلمة "تقنية" بتأثير gradient
- **Shimmer animation** على الكلمة المميزة
- **وصف** توضيحي تحت العنوان

### 3️⃣ **الصورتان الجانبيتان**

#### الصورة الأولى (1.jpeg):
- على اليمين
- تأثير Hover مع zoom
- Overlay يظهر عند الـ hover:
  - أيقونة "فريق خبراء"
  - عنوان ووصف
- Glow effect متحرك (Shine animation)
- Shadow مع تأثير برتقالي

#### الصورة الثانية (Capital Market):
- على اليسار
- نفس التأثيرات
- Overlay مختلف:
  - أيقونة صاروخ
  - "حلول مبتكرة"
  - "قيادة الثورة الرقمية"

### 4️⃣ **الخدمات في المنتصف (6 بطاقات)**

بطاقات الخدمات داخل صندوق Glassmorphism:
1. **تطوير البرمجيات** (fa-code)
2. **البنية التحتية** (fa-cloud)
3. **الأمن السيبراني** (fa-shield-alt)
4. **تحليل البيانات** (fa-chart-line)
5. **تطبيقات الجوال** (fa-mobile-alt)
6. **الذكاء الاصطناعي** (fa-robot)

كل بطاقة:
- أيقونة بـ gradient برتقالي
- عنوان ووصف
- Hover effect مع:
  - تحريك لليسار
  - تغيير الخلفية
  - Shine animation
  - تكبير الأيقونة مع دوران

### 5️⃣ **زر CTA**
- لون أخضر واتساب
- أيقونة واتساب + نص + سهم
- Hover:
  - يرتفع للأعلى
  - يكبر قليلاً
  - Shine effect
  - السهم يتحرك

## 💫 التأثيرات الحركية

### Animations المستخدمة:
1. **float** - للدوائر الخلفية (20s)
2. **badgeGlow** - للـ badge (2s)
3. **pulse** - لأيقونة الـ badge (2s)
4. **shimmer** - للكلمة المميزة (3s)
5. **shine** - للصور (3s)
6. **slide** - عند hover على الخدمات

### Hover Effects:
- ✅ الصور: zoom + overlay + shadow
- ✅ الخدمات: slide left + glow + icon rotate
- ✅ الزر: lift + scale + shine

### Small Mobile (< 480px):

- text-align: center
- padding أصغر
